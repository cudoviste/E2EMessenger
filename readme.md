## Svelte Socket.io e2e encrypted web messenger tutorial
___

clone this repo

```shell
git clone https://github.com/cudoviste/e2emessenger.git proje
```

### build client app

go client folder

```shell
cd ./proje/client

```

install dependencies

```node
// install with npm
npm install
// or with yarn 
yarn install
```

build app
```node
// build with npm
npm run build

// or with yarn
yarn build
```

## prepare server

```shell
cd ../server
```

install dependencies

```node
// install with npm
npm install
// or with yarn 
yarn install
```

run server deamon

```node
node index.js
```


now you can visit http://localhost:3000 and play with the app
